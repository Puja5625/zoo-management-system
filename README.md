# Zoo Management System (Laravel 10)

A professional web application built to manage zoo operations, including animal tracking, staff management, and ticketing systems.

## 🚀 Features
- **Animal Management:** CRUD operations for animal profiles with category filtering.
- **Staff Management:** Role-based access control for zookeepers and admins.
- **Ticketing System:** Generate and manage visitor tickets.
- **Reporting:** Simple dashboard for daily zoo statistics.

## 🛠 Tech Stack
- **Backend:** PHP 8.1+ & Laravel 10
- **Frontend:** Blade Templates & Tailwind CSS / Bootstrap
- **Database:** MySQL
- **Authentication:** Laravel Breeze / Fortify (သုံထားလျှင် ထည့်ရေးရန်)

## 📋 Installation
1. Clone the repository: `git clone [your-link]`
2. Install dependencies: `composer install` & `npm install`
3. Set up environment: `cp .env.example .env`
4. Generate key: `php artisan key:generate`
5. Run migrations: `php artisan migrate --seed`
6. Start server: `php artisan serve`
